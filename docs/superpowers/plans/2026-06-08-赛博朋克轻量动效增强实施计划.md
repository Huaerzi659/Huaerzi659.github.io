# 赛博朋克轻量动效增强 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在现有暗色霓虹博客 UI 上增加克制的赛博朋克动效，让首页和卡片更有在线工作台氛围。

**Architecture:** 只使用 Astro 组件内 CSS 和全局 CSS，不新增 JavaScript 或依赖。全局文件提供 motion token 和通用动画，首页/卡片/标签/Header 各自使用局部选择器增强交互。验证通过构建、RSS 检查、静态 CSS 检查和浏览器截图完成。

**Tech Stack:** Astro、CSS、GitHub Pages。

---

### Task 2: Global motion tokens and page entry

**Files:**
- Modify: `src/styles/global.css`

- [ ] **Step 1: Add motion tokens**

Add:

```css
--ease-out: cubic-bezier(0.16, 1, 0.3, 1);
--motion-short: 180ms;
--motion-medium: 420ms;
```

- [ ] **Step 2: Add keyframes**

Add `panel-enter`, `signal-pulse`, and `sheen-pass` keyframes.

- [ ] **Step 3: Add no-preference animation rules**

Inside `@media (prefers-reduced-motion: no-preference)`, animate `.surface`, `.article-card`, `.tag-pill`, and `.desk-panel` with `panel-enter`.
