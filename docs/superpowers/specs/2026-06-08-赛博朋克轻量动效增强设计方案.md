# 赛博朋克轻量动效增强设计方案

## 背景

当前博客已经是暗色霓虹赛博朋克风格，首页有工作台 hero、文章卡片、主题 chip 和扫描线背景。下一步不是重做视觉，而是在已有方向上增加一点“活着的界面”感，让页面更像一个在线工作台。

## 目标

- 增加轻量、克制的动态效果。
- 保持个人博客阅读优先，不让动画抢正文注意力。
- 不引入 JavaScript 和新依赖。
- 尊重 `prefers-reduced-motion`。
- 不使用横向滚动、滚动劫持、夸张弹跳或无限大面积装饰动画。
- 保持现有暗色霓虹视觉，不切回浅色或杂志风。

## 方案

采用方案 1：轻量氛围增强。

主要变化：

- 全局增加统一动效 token，例如 `--ease-out`、`--motion-short`。
- 首页 hero 的身份面板增加 HUD 边框、淡扫描光带和角标感。
- 右侧工作台卡片增加非常轻的入场动画。
- `status-dot` 增加低强度呼吸效果。
- 文章卡片 hover 增加细线扫光和更明确的激活状态。
- 可点击主题 chip hover 增加短扫光。
- Header 品牌 mark 加轻微 hover 激活感。

## 可访问性

- 所有连续或入场动画都放在 `@media (prefers-reduced-motion: no-preference)` 内。
- 现有 `prefers-reduced-motion: reduce` 保持兜底，关闭动画和长过渡。
- 动画不承载必要信息；不依赖 hover 才能看到核心内容。
- 移动端不新增会造成横向滚动的固定宽度元素。

## 验证

- Astro build 成功。
- RSS 验证仍通过。
- 新增静态验证脚本检查关键动效选择器和 reduced-motion 保护。
- 使用浏览器检查首页桌面与移动端没有明显重叠、空白或横向滚动。
