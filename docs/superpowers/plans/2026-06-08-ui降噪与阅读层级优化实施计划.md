> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Tech Stack:** Astro、CSS、原生验证脚本、GitHub Pages。

### Task 2: Global Noise Reduction

**Files:**
- Modify: `src/styles/global.css`

- [ ] **Step 1: Add calmer tokens**

Add these tokens to `:root`:

```css
--copy: #c9d8e8;
--surface-quiet: rgba(8, 12, 26, 0.72);
--surface-muted: rgba(11, 16, 32, 0.66);
--line-muted: rgba(155, 178, 199, 0.14);
--glow-soft: 0 0 18px rgba(0, 245, 255, 0.1);
--scanline-opacity: 0.14;
```

- [ ] **Step 2: Reduce global decoration strength**

Use `--scanline-opacity` for `body::before`, lower the radial background opacity, make `.surface` use `--surface-quiet`, and reduce default `--shadow` strength.

- [ ] **Step 3: Reduce entry animation density**

In the `prefers-reduced-motion: no-preference` block, keep entry animation only for `.desk-panel`, `.topic-panel`, and `.footer-panel`, not every tag and article card.
